# Projeto para aula-2 de React & Next.js

Este é um projeto [Next.js](https://nextjs.org) gerado a partir de [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Passos para utilizar este projeto

1. Clique no link do repositório GitHub https://github.com/ULHT-PW/next-aula-02
2. Clique em Fork
4. Abra no Codespace
5. Execute `nvm use 20`
6. Corra o servidor de desenvolvimento com `npm run dev`
9. Abra [http://localhost:3000](http://localhost:3000) com o seu browser para ver a aplicação.
10. Garanta que tem File/AutoSave ativado
11. faça deploy na [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme)
